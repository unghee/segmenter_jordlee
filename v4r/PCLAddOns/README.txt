PCL Add Ons
===========

Requirements:
-------------
- OpenCV
- PCL


Implementations:
----------------
- BilateralFilter: Filter point cloud and smooth surfaces
- CCEuclideanClustering: Cross-component euclidean clustering
- CCLabeling: Cross-component labeling
- ContourDetection: ???
- EuclideanClustering: Euclidean clustering of point clouds
- ModelFitter: Fitting different shape models (plane, cylinder, sphere) in point cloud.
- NormalsClustering: ???
- NormalsEstimationNR: ???
- PCLFunctions: Several functions used with PCL library
- PCLUtils: Mainly convert functions or "little helpers"
- PlanePopout (with dominant plane, ROI and SOI calculation)
- SubsamplePointCloud: Resize a point cloud
